# 8-Bit Temple-Run-Style iOS Mini Game

This folder contains a **simple endless runner prototype** built with **SwiftUI + SpriteKit**.

## Features
- 3-lane runner movement (swipe left/right)
- Tap-to-jump controls
- Randomly spawning obstacles
- Increasing score over time
- Retro 8-bit style pixel sprites (generated in code)
- Game-over + tap-to-restart loop

## How to run
1. Open Xcode and create a new iOS App project (SwiftUI lifecycle).
2. Add these files to your project:
   - `RetroRunnerApp.swift`
   - `ContentView.swift`
   - `GameScene.swift`
3. Ensure your deployment target is iOS 16+.
4. Build and run on simulator/device.

## Notes
This is intentionally lightweight so you can expand it quickly with:
- coin pickups
- power-ups
- custom sprite sheets
- procedural level themes
- chiptune soundtrack
