# Castle Kickflip GB

A tiny Game Boy-inspired skateboarding game set inside a castle.

## Play on iOS

1. Open `index.html` from a static host (for example GitHub Pages, Netlify, or any web server).
2. On iPhone/iPad in Safari, open the game URL.
3. Tap **Share → Add to Home Screen** for full-screen play.

## Controls

- **Left / Right**: move
- **Jump**: jump
- **Grind (Z)**: grind in air to avoid some hits
- **Trick (X)**: do a trick in air for bonus score
- **Restart**: restart after wipeout
